# Task Management System

A full-stack Task Management System built using Django, Django REST Framework, and React.
This application helps users create, manage, update, and track daily tasks efficiently.

---

## 🚀 Features

* User Authentication
* Create, Update, Delete Tasks
* Task Status Management
* REST API Integration
* Responsive UI
* Dashboard for Task Tracking
* Secure Backend APIs
* Real-time Task Updates
